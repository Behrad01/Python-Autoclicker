# Python-Autoclicker
Simple autoclicker in python, made this by myself, this can be used in games and other tasks related to clicking.

## Note
- This script accept hotkeys but not special keys as hotkeys

# How to run
run this script:
```bash
python autoclicker.py
```
make sure to have python installed and then run this script in CMD (Command Prompt).
